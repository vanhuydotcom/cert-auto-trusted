Certificate Auto-Trust Tool
============================

This installer will automatically trust an SSL certificate on your Windows system.

What this does:
- Installs a pre-configured SSL certificate to your Windows certificate store
- Adds the certificate to Trusted Root Certification Authorities
- Enables HTTPS websites using this certificate to work without security warnings
- Works with Chrome, Edge, and other browsers that use Windows certificate store

Requirements:
- Windows 10 or later
- Administrator privileges (required for certificate installation)

After installation:
- The certificate will be automatically trusted by Windows
- You can verify by visiting your HTTPS website in Chrome or Edge
- No security warnings should appear

Security Note:
- Only install certificates from sources you trust
- This certificate will be trusted system-wide
- You can uninstall the application later, but the certificate will remain in the store
- To remove the certificate, use Windows Certificate Manager (certmgr.msc)

For more information, see the README.md file in the installation directory.
